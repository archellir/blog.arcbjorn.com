Quick Local Test

Run your development server and visit these URLs:

  • http://localhost:8000/ - Check homepage meta tags
  • http://localhost:8000/bare-metal-k8s-production - Check a blog post
  • http://localhost:8000/sitemap.xml - Verify sitemap

Then view page source and look for:

  • <meta property="og:image" - Should show your image URL
  • <meta name="twitter:card" - Should be "summary_large_image"
  • <script type="application/ld+json" - Structured data
